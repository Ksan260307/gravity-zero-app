# GRAVITY ZERO — 3D フライトシューター

無重力空間の「気持ちいい慣性操作」を体験する3Dフライトシューターです。
ブラウザでそのまま動作し、Capacitor を使って iOS / Android アプリとしても公開できます。

## 特徴

- **スコアと難易度** — 敵を倒すとスコアが入り、1500点ごとにレベルが上昇。敵の出現数・突入速度・耐久が段階的に上がります
- **耐久ゲージ** — 敵と接触するとダメージ。0になるとゲームオーバーで、その場で再スタートできます
- **サブ武器(アイテムで入手、メインショットと同時使用可)**
  - **ハルバード** — 押している間、太い貫通レーザーを前方へ照射(連続5秒で自動停止)。クールタイムは使った時間に比例し、最短1秒〜最長5秒。照射中はゲージが残り時間を表示
  - **コメット** — 単発の誘導弾。1秒のクールタイム
  - **ホーミングミサイル** — 10発を上下へ撒いてから目標を追尾。10秒のクールタイム
- **3タイプの敵** — まっすぐ突っ込む「突撃型」、左右へ大きく蛇行する「蛇行型」、遠距離から狙撃してくるオレンジの「砲撃型」(レベル2以降)が混ざって出現します
- **ボス戦** — 2000点で最初の大型機が出現し、以降は撃破ごとに3000点間隔で再来。倒すと約1.5秒の断末魔演出(連鎖爆発→大爆発+衝撃波リング)とともに1000点ボーナス+修理アイテムを確定ドロップ
- **個性ある3つの機体モード** — LIGHT/STANDARD/HEAVYを切り替えると、質量・空間抵抗に加えて **防御力・連射速度・火力** が変化し、立ち回りそのものが変わります
- **チャレンジモード** — 無敵+2分の制限時間でスコアだけを競うモード。敵や敵弾はすり抜けるので撃って倒すしかありません。タイトルの「CHALLENGE MODE」から開始でき、ベストは通常モードと別枠で保存されます
- **ローカルランキング** — 通常モードの上位5スコアを自動保存し、ゲームオーバー画面に表示。今回のスコアは強調表示されます
- **開始レベル選択** — タイトルの「START LEVEL」で 1 / 3 / 5 から開始難易度を選べます(腕試しに)
- **キーコンフィグ** — ヘルプ画面(？ / H)からPCの操作キーを自由に変更可能。重複キーは自動で入れ替え、設定はブラウザに保存されます
- **一斉ロックオン + 追尾レーザー** — 長押しで前方の敵を最大18機まで捕捉し、離すとロック数に関わらず**固定24本**のレーザーが対象へ振り分けられて飛びます(1本の威力は控えめ)。少数ロックには弾が集中し、多数ロックには広く行き渡ります。発射後の硬直はごく短く、テンポを崩しません
- **ブースト** — 方向キーを押している向きへ高速ダッシュ(通常の約3.6倍の加速)、何も押さなければ前方へ突進します。押している間だけ効くシンプルな加速で、無敵やゲージ制限はありません
- **ゼロシフト(短距離瞬間移動)** — 回避と攻めのアクセント
- **ショット進化** — 15機撃破でツインショット(横2列)、40機撃破でトリプルショット(横3列)へ主砲が強化されます。被弾しても失われません(リスタートで初期状態へ)
- **ウィスプ(支援ビット)** — 100機撃破ごとに機体へ追従する小型ビットを1機装備(最大3機)。各ウィスプは1秒ごとに前方へ単発弾を撃ち、手数を底上げします
- **連続撃破コンボ** — 2.5秒以内に撃破をつなぐとスコアボーナスが伸びます(最大+100%)。被弾で途切れるため、無敵のチャレンジモードではコンボ維持がスコアアタックの鍵になります
- **グレイズ** — 敵すれすれを通過すると+15点(敵1機につき1回)。ゼロシフトやブーストで敵の間を縫う、攻めた機動が報われます
- **NEW RECORD 表示** — ベストスコアを更新するとゲームオーバー画面で祝福されます
- **修理アイテムとハイスコア** — ドロップの30%は耐久を30回復する修理アイテム。近づくと自機へ吸い寄せられるので取り逃しません
- **スタイリッシュな英語HUD** — プレイ中の表示は英語表記の1パネル(SCORE / LV / DEF / ROF / PWR / SUB)。操作説明・キー設定などの説明画面は日本語のままです
- **デバッグモード** — F2キー(または URL に `?debug`)で開発用パネルを表示。無敵化・ボス即出現・スコア加算・FPS/オブジェクト数の確認ができます
- **加速感の演出** — 急な切り返しに反応する集中線・視野角の揺れ・カメラの遅れ・被弾の強さに応じた画面シェイクなど、体感重視の演出
- **外部アセット不要** — 効果音・BGMはすべてWeb Audio APIでリアルタイム合成。画像ファイルもありません

## 操作方法

| 操作 | PC | スマホ |
|---|---|---|
| 移動 | W/A/S/D または矢印キー | 左の仮想スティック |
| ショット | SPACE | ショットボタン |
| サブ武器 | R(アイテムで入手) | サブ武器ボタン |
| 一斉ロックオン | F 長押し → 離して発射 | ロックボタン |
| ブースト(方向ダッシュ / 前方突進) | E 長押し(+ 移動方向) | BOOSTボタン |
| ゼロシフト(瞬間移動) | Q + 移動方向 | ZERO SHIFTボタン |
| モード切替 | SHIFT | MODEボタン |
| ヘルプ / 一時停止 | H | 画面右上の「？」ボタン |

## 機体モードの個性

| モード | 防御 | 連射 | 火力 | 特徴 |
|---|---|---|---|---|
| LIGHT(軽量) | LOW | FAST | MID | 俊敏で慣性が残らない。手数は多いが被弾に弱い |
| STANDARD(標準) | MID | MID | MID | 防御・火力・機動のバランス型 |
| HEAVY(重装) | HIGH | SLOW | HIGH | 装甲が厚く被弾に強い。連射は遅いが一撃が重い |

被弾ダメージは軽量が1.6倍、重装が0.5倍。通常ショットは重装が1発で2ダメージを与えます。

## ルール

- 敵を撃破: 100点(大型は200点)。2.5秒以内の連続撃破でコンボボーナス(最大2倍) / グレイズ(すれすれ回避): 15点 / アイテム取得: 300点(修理は100点) / ボス撃破: 1000点
- 敵と接触すると耐久25、ボス弾は15のダメージ(モードの装甲で増減)。被弾直後は短時間無敵(機体が点滅)
- 撃破した敵はときどきアイテム(光る輪)を落とします。サブ武器は取ると即装備で入れ替え、緑の修理アイテムは耐久を30回復します
- スコア1500点ごとにレベルが上がり、敵の出現数・速度・耐久が増加。2000点で最初のボスが出現します

## ブラウザで動かす

`www/index.html` は ES Modules を使っているため、`file://` では開けません。
簡易サーバーを起動してアクセスしてください。

```sh
# Python がある場合
python -m http.server 8000 --directory www
# → http://localhost:8000 を開く

# Node.js がある場合
npx serve www
```

## スマホアプリとして公開する

[Capacitor](https://capacitorjs.com/) でWebアプリをそのままネイティブアプリ化します。

### 事前準備

- [Node.js](https://nodejs.org/)(LTS版)
- Android: [Android Studio](https://developer.android.com/studio)
- iOS: macOS + [Xcode](https://developer.apple.com/xcode/)(iOSビルドはMacでのみ可能)

### 手順

```sh
cd gravity-zero-app
npm install

# Android
npm run android:add     # 初回のみ:androidフォルダを生成
npm run sync            # www の内容をネイティブ側へ反映
npm run android:open    # Android Studio で開く → 実機で実行 / AABをビルド

# iOS (Macのみ)
npm run ios:add
npm run sync
npm run ios:open        # Xcode で開く → 実機で実行 / Archive
```

`www/` 内を変更したら、そのつど `npm run sync` で反映してください。

### ストア公開前のチェックリスト

1. **アプリID** — `capacitor.config.json` の `appId` を自分のドメインに変更する(例: `jp.yourname.gravityzero`)。ストア公開後は変更できません
2. **アイコンとスプラッシュ** — `npx @capacitor/assets generate` で `assets/icon.png`(1024×1024)から各サイズを自動生成できます
3. **署名** — Android は keystore で署名した AAB を Google Play Console へ、iOS は Apple Developer Program($99/年)のアカウントで App Store Connect へ提出します
4. **画面向き** — 横持ち推奨のため、必要に応じてネイティブ側で landscape 固定にしてください
   - Android: `AndroidManifest.xml` の activity に `android:screenOrientation="landscape"`
   - iOS: Xcode の Deployment Info で Landscape のみチェック

## 技術メモ

- 描画は Three.js(WebGL)。ライブラリはローカル同梱のためオフラインでも動作します
- 弾・敵・レーザー・粒子はすべて固定長プールで再利用し、プレイ中のメモリ確保を避けています
- 機体の運動は「推力 ÷ 質量 − 速度比例の抵抗」の単純な積分で、モード切替は質量と抵抗の変更として実装しています
- 加速度の急変(切り返し・発射・被弾)を検出して、集中線や視野角の揺れなどの演出に変換しています
- 遊びやすさの調整:自機の当たり判定は見た目より小さめ(半径9)。敵は自機の手前(z > -120)でホーミングをやめて軌道を確定させるため、進路を読んで回避できます。硬直中も慣性が残り、壁の反発は弱めです

本作は『**ANUBIS ZONE OF THE ENDERS**』の高速機動戦へのリスペクトを込めて制作した、ファンメイドの習作です。一斉ロックオンからの追尾レーザー、ゼロシフト、質量が変わる機体モードなどの操作感は、同作の「気持ちよさ」を独自に解釈して再構成したものです。
