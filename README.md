# GRAVITY ZERO — 3D フライトシューター

無重力空間の「気持ちいい慣性操作」を体験する3Dフライトシューターです。
ブラウザでそのまま動作し、Capacitor を使って iOS / Android アプリとしても公開できます。

## 特徴

- **スコアと難易度** — 敵を倒すとスコアが入り、1500点ごとにレベルが上昇。敵の出現数・突入速度・耐久が段階的に上がります
- **耐久ゲージ** — 敵と接触するとダメージ。0になるとゲームオーバーで、その場で再スタートできます
- **サブ武器(アイテムで入手、メインショットと同時使用可)**
  - **ハルバード** — 押している間、太い貫通レーザーを前方へ照射(連続5秒で自動停止)。クールタイムは使った時間に比例し、最短1秒〜最長5秒。照射中はゲージが残り時間を表示
  - **コメット** — 単発の誘導弾。1秒のクールタイム
  - **ホーミングミサイル** — 10発を上下へ撒いてから目標を追尾。10秒のクールタイム
- **ボス戦** — 2000点で最初の大型機が出現し、以降は撃破ごとに3000点間隔で再来。自機を狙う3方向弾を撃ち、倒すと1000点ボーナス+修理アイテムを確定ドロップ。ロックオンや全サブ武器の対象にもなります
- **個性ある3つの機体モード** — 軽量/標準/重装を切り替えると、質量・空間抵抗に加えて **防御力・連射速度・火力** が変化し、立ち回りそのものが変わります
- **キーコンフィグ** — ヘルプ画面(？ / H)からPCの操作キーを自由に変更可能。重複キーは自動で入れ替え、設定はブラウザに保存されます
- **一斉ロックオン + 追尾レーザー** — 長押しで前方の敵を最大18機まで捕捉し、離すと曲線を描くレーザーが一斉に飛びます
- **ブースト / ブリンク(短距離瞬間移動)** — 回避と攻めのアクセント
- **修理アイテムとハイスコア** — ドロップの30%は耐久を30回復する修理アイテム。ベストスコアは自動保存されゲームオーバー画面に表示されます
- **シンプルなHUD・タイトル** — プレイ中の表示はスコア・耐久・モード・サブ武器の1パネルのみ。タイトル画面も要点3行+ベストスコアだけの構成で、操作説明とキー設定はヘルプ画面(？ / H)に集約。タブを離れると自動で一時停止します
- **加速感の演出** — 急な切り返しに反応する集中線・視野角の揺れ・カメラの遅れなど、体感重視の演出
- **外部アセット不要** — 効果音・BGMはすべてWeb Audio APIでリアルタイム合成。画像ファイルもありません

## 操作方法

| 操作 | PC | スマホ |
|---|---|---|
| 移動 | W/A/S/D または矢印キー | 左の仮想スティック |
| ショット | SPACE | ショットボタン |
| サブ武器 | R(アイテムで入手) | サブ武器ボタン |
| 一斉ロックオン | F 長押し → 離して発射 | ロックボタン |
| ブースト | E 長押し | ブーストボタン |
| ブリンク(瞬間移動) | Q + 移動方向 | 瞬間移動ボタン |
| モード切替 | SHIFT | モードボタン |
| ヘルプ / 一時停止 | H | 画面右上の「？」ボタン |

## 機体モードの個性

| モード | 防御 | 連射 | 火力 | 特徴 |
|---|---|---|---|---|
| 軽量 | 弱 | 速 | 標準 | 俊敏で慣性が残らない。手数は多いが被弾に弱い |
| 標準 | 標準 | 標準 | 標準 | 防御・火力・機動のバランス型 |
| 重装 | 強 | 遅 | 高 | 装甲が厚く被弾に強い。連射は遅いが一撃が重い |

被弾ダメージは軽量が1.6倍、重装が0.5倍。通常ショットは重装が1発で2ダメージを与えます。

## ルール

- 敵を撃破: 100点(大型は200点) / アイテム取得: 300点(修理は100点) / ボス撃破: 1000点
- 敵と接触すると耐久25、ボス弾は15のダメージ(モードの装甲で増減)。被弾直後は短時間無敵(機体が点滅)
- 撃破した敵はときどきアイテム(光る輪)を落とします。サブ武器は取ると即装備で入れ替え、緑の修理アイテムは耐久を30回復します
- スコア1500点ごとにレベルが上がり、敵の出現数・速度・耐久が増加。2000点で最初のボスが出現します

## テスト

ゲーム本体を実際に読み込み、フライト挙動・モード個性・全サブ武器(照射上限・比例クールタイム含む)・ボス戦・キーコンフィグ・スコア/難易度・被弾/ゲームオーバー・アイテム取得・ハイスコア保存・一時停止・UI/タイトル簡素化など全機能を検証する統合テストを同梱しています(146項目)。

**アプリのルート**(`www` の1つ上)を配信して `tests/test.html` を開くと、結果が画面に表示されます。

```sh
# gravity-zero-app/ をそのまま配信する
python -m http.server 8000
# → http://localhost:8000/tests/test.html を開く
```

全項目が緑(成功)になれば正常です。要約は `window.__testResults`(`{ total, passed, failed, failures }`)にも入るため、自動化からも参照できます。

## ブラウザで動かす

`www/index.html` は ES Modules を使っているため、`file://` では開けません。
簡易サーバーを起動してアクセスしてください。

```sh
# Python がある場合
python -m http.server 8000 --directory www
# → http://localhost:8000 を開く

# Node.js がある場合
npx serve www
```

## スマホアプリとして公開する

[Capacitor](https://capacitorjs.com/) でWebアプリをそのままネイティブアプリ化します。

### 事前準備

- [Node.js](https://nodejs.org/)(LTS版)
- Android: [Android Studio](https://developer.android.com/studio)
- iOS: macOS + [Xcode](https://developer.apple.com/xcode/)(iOSビルドはMacでのみ可能)

### 手順

```sh
cd gravity-zero-app
npm install

# Android
npm run android:add     # 初回のみ:androidフォルダを生成
npm run sync            # www の内容をネイティブ側へ反映
npm run android:open    # Android Studio で開く → 実機で実行 / AABをビルド

# iOS (Macのみ)
npm run ios:add
npm run sync
npm run ios:open        # Xcode で開く → 実機で実行 / Archive
```

`www/` 内を変更したら、そのつど `npm run sync` で反映してください。

### ストア公開前のチェックリスト

1. **アプリID** — `capacitor.config.json` の `appId` を自分のドメインに変更する(例: `jp.yourname.gravityzero`)。ストア公開後は変更できません
2. **アイコンとスプラッシュ** — `npx @capacitor/assets generate` で `assets/icon.png`(1024×1024)から各サイズを自動生成できます
3. **署名** — Android は keystore で署名した AAB を Google Play Console へ、iOS は Apple Developer Program($99/年)のアカウントで App Store Connect へ提出します
4. **画面向き** — 横持ち推奨のため、必要に応じてネイティブ側で landscape 固定にしてください
   - Android: `AndroidManifest.xml` の activity に `android:screenOrientation="landscape"`
   - iOS: Xcode の Deployment Info で Landscape のみチェック

## 技術メモ

- 描画は Three.js(WebGL)。ライブラリはローカル同梱のためオフラインでも動作します
- 弾・敵・レーザー・粒子はすべて固定長プールで再利用し、プレイ中のメモリ確保を避けています
- 機体の運動は「推力 ÷ 質量 − 速度比例の抵抗」の単純な積分で、モード切替は質量と抵抗の変更として実装しています
- 加速度の急変(切り返し・発射・被弾)を検出して、集中線や視野角の揺れなどの演出に変換しています
